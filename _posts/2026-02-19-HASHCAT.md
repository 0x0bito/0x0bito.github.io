## Tổng Quan

> Hashcat **không giải mã** hash — nó thử hàng triệu/tỷ mật khẩu, hash chúng lại, rồi so sánh với hash mục tiêu. Khi khớp → tìm được mật khẩu gốc.
{: .prompt-info }

---

## Cú Pháp Cơ Bản

```bash
hashcat -m <hash_type> -a <attack_mode> <hash_file> <wordlist/mask>
```

| Tham số | Ý nghĩa |
|---|---|
| `-m` | Loại hash (hash type ID) |
| `-a` | Chế độ tấn công (attack mode) |
| `-o` | File lưu kết quả |
| `--show` | Hiển thị hash đã crack |
| `--username` | File hash có chứa username |
| `-r` | Áp dụng rule file |
| `--force` | Bỏ qua cảnh báo (dùng trên VM) |
| `-w 3` | Workload profile (1-4) |
| `--status` | Hiển thị trạng thái trong lúc chạy |

## Các Loại Hash Phổ Biến (`-m`)

### Hash Tổng Quát (Generic)

| ID | Hash Type | Prefix / Độ dài | Ghi chú |
|---|---|---|---|
| `0` | MD5 | 32 hex | Rất phổ biến, nhanh crack |
| `100` | SHA-1 | 40 hex | Git, cũ |
| `1400` | SHA-256 | 64 hex | |
| `1700` | SHA-512 | 128 hex | |
| `17300` | Keccak-256 | 64 hex | Tiền thân SHA3 |

### Linux / Unix

| ID | Hash Type | Prefix | Ghi chú |
|---|---|---|---|
| `500` | md5crypt | `$1$` | `/etc/shadow` cũ |
| `1800` | sha512crypt | `$6$` | `/etc/shadow` hiện đại |
| `7400` | sha256crypt | `$5$` | `/etc/shadow` |
| `400` | phpass | `$P$` / `$H$` | WordPress, phpBB |

### Windows / Active Directory

| ID | Hash Type | Ghi chú |
|---|---|---|
| `1000` | NTLM | SAM, secretsdump, Mimikatz |
| `3000` | LM | Windows cũ (< Vista) |
| `5500` | NetNTLMv1 | Responder (cũ) |
| `5600` | NetNTLMv2 | Responder capture (phổ biến) |
| `13100` | Kerberoast TGS-REP | AD — GetUserSPNs.py |
| `18200` | AS-REP Roast | AD — no preauth |
| `2100` | DCC2 (MSCACHEv2) | Domain cached credentials |

### Web Application

| ID | Hash Type | Prefix | Ghi chú |
|---|---|---|---|
| `3200` | bcrypt | `$2a$` / `$2b$` | Chậm, web app hiện đại |
| `400` | phpass | `$P$` | WordPress |

### Network / Services

| ID | Hash Type | Ghi chú |
|---|---|---|

## Các Chế Độ Tấn Công (`-a`)

### Mode 0 — Dictionary Attack

```bash
hashcat -m 0 -a 0 hash.txt /usr/share/wordlists/rockyou.txt
```

### Mode 1 — Combination Attack

```bash
# "pass" + "word" → "password"
hashcat -m 0 -a 1 hash.txt wordlist1.txt wordlist2.txt
```

### Mode 3 — Brute-force / Mask Attack

```bash
# 8 ký tự chữ thường
hashcat -m 0 -a 3 hash.txt ?l?l?l?l?l?l?l?l

# 4 chữ hoa + 4 số
hashcat -m 0 -a 3 hash.txt ?u?u?u?u?d?d?d?d

# Increment 6-8 ký tự
hashcat -m 0 -a 3 --increment --increment-min=6 hash.txt ?a?a?a?a?a?a?a?a
```

**Ký hiệu Mask:**

| Ký hiệu | Bộ ký tự |
|---|---|
| `?l` | a-z (chữ thường) |
| `?u` | A-Z (chữ hoa) |
| `?d` | 0-9 (số) |
| `?s` | Ký tự đặc biệt `!@#$...` |
| `?a` | Tất cả (`?l?u?d?s`) |
| `?h` | Hex thường `0-9a-f` |
| `?H` | Hex hoa `0-9A-F` |

### Mode 6 — Hybrid Wordlist + Mask

```bash
# "password" + "123" → "password123"
hashcat -m 0 -a 6 hash.txt rockyou.txt ?d?d?d
```

### Mode 7 — Hybrid Mask + Wordlist

```bash
# "2024" + "password" → "2024password"
hashcat -m 0 -a 7 hash.txt ?d?d?d?d rockyou.txt
```

## Rules

```bash
# Rule có sẵn
hashcat -m 0 -a 0 hash.txt rockyou.txt -r /usr/share/hashcat/rules/best64.rule

# Nhiều rule
hashcat -m 0 -a 0 hash.txt rockyou.txt -r rules/best64.rule -r rules/d3ad0ne.rule
```

**Rule files phổ biến:**

| Rule File | Mô tả |
|---|---|
| `best64.rule` | 64 rule hiệu quả nhất |
| `rockyou-30000.rule` | Rule từ phân tích rockyou |
| `d3ad0ne.rule` | Biến thể phức tạp |
| `OneRuleToRuleThemAll.rule` | Rule tổng hợp lớn |

**Tự tạo rule:**

```bash
# rule.txt
l           # lowercase toàn bộ
u           # uppercase toàn bộ
c           # Capitalize chữ đầu
r           # Reverse: "password" → "drowssap"
$1          # Thêm "1" vào cuối
^2024       # Thêm "2024" vào đầu
sa@         # Thay 'a' → '@'

hashcat -m 0 -a 0 hash.txt rockyou.txt -r rule.txt
```

## Ví Dụ Thực Tế

### Crack NTLM (Windows)

```bash
hashcat -m 1000 -a 0 ntlm_hashes.txt rockyou.txt
hashcat -m 1000 -a 0 --username ntlm_hashes.txt rockyou.txt
```

### Crack /etc/shadow (Linux)

```bash
unshadow /etc/passwd /etc/shadow > combined.txt
hashcat -m 1800 -a 0 combined.txt rockyou.txt
```

### Crack NetNTLMv2 (từ Responder)

```bash
hashcat -m 5600 -a 0 netntlmv2.txt rockyou.txt
```

### Crack Kerberoast TGS

```bash
hashcat -m 13100 -a 0 kerberoast.txt rockyou.txt -r rules/best64.rule
```

### Crack WPA2 WiFi

```bash
hashcat -m 22000 -a 0 wifi.hc22000 rockyou.txt
```

### Crack bcrypt

```bash
hashcat -m 3200 -a 0 bcrypt.txt rockyou.txt -w 3
```

## Tối Ưu Hiệu Suất

```bash
# Workload cao nhất
hashcat -m 0 -a 0 hash.txt rockyou.txt -w 4

# Force CPU (VM không có GPU)
hashcat -m 0 -a 0 hash.txt rockyou.txt --force
```

> Trên VM (VirtualBox/VMware), GPU thường không khả dụng. Phải dùng `--force` để ép dùng CPU — tốc độ sẽ chậm hơn nhiều.
{: .prompt-warning }

## Quản Lý Session & Restore

```bash
# Đặt tên session
hashcat -m 0 -a 0 hash.txt rockyou.txt --session mysession

# Resume
hashcat --restore --session mysession
```

## Nhận Diện Hash

```bash
hashid '$2a$12$xyz...'
hash-identifier
hashcat --example-hashes | grep -A2 "NTLM"
```

> Nhận diện nhanh qua prefix:
> - `$1$` → MD5crypt (500)
> - `$2a$` / `$2b$` → bcrypt (3200)
> - `$5$` → SHA256crypt (7400)
> - `$6$` → SHA512crypt (1800)
> - `*` đầu dòng → MySQL (300)
> - 32 hex không prefix → MD5 (0)
> - 40 hex không prefix → SHA1 (100)
{: .prompt-tip }

## Lỗi Thường Gặp

| Lỗi | Nguyên nhân | Giải pháp |
|---|---|---|
| `No devices found` | Không nhận GPU | Thêm `--force` dùng CPU |
| `Token length exception` | Hash format sai | Kiểm tra lại định dạng hash |
| `Separator unmatched` | Hash bị lỗi dòng | Dùng `--username` nếu có prefix |
| `Exhausted` | Hết wordlist, không tìm ra | Đổi wordlist hoặc thêm rule |
| `Cracked` | Tìm được! | Xem kết quả bằng `--show` |
