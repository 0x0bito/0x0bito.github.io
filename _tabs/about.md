## Hi, I'm Obito aka Tobi

- 🔴 Offensive Security enthusiast
- 🐛Penetration Tester & Security Researcher

## About Me

Passionate about offensive security and penetration testing. My research focuses on
finding and responsibly disclosing vulnerabilities in web applications — covering
SQL Injection, Broken Access Control, IDOR, and Remote Code Execution.

This blog is where I document my research, notes, and technical writeups along the way.

> *"Break things. Understand why. Document everything."*
